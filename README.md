# artlift
Artlift project cheer up some artists. We would like them to make strong connection between industoties and their funs. Artists can upload their artwoks and make their social media. People can interact them and make his or her fan. They will cheer up by purchasing their products or joining their subscription plans.

## **Features**
- Artist Information: artist logo, features, passions, popular artworks and so on
- Artwork: Slider pictures, discriptions of artworks, title, Comment interactions
- Searching artists and artworks
- Admin page:manage profile, upload artworks, manage profits, shipping places etc.
- Purchasing page

## **Technology Stack**
- **Backend**: Django with Django REST Framework (DRF)
  - Handles API endpoints for 
  - Supabase
- **Frontend**: Nextjs, Typescript
  - Display user data, artworks, handles filtering, and manages user interactions.
  - Integrated with backend APIs.
 

## **How to start frontend
1. **Navigate to the `frontend/` Directory**:
   ```bash
   cd frontend/artlift-client
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run npm**:
   ```bash
   npm run dev
   ```

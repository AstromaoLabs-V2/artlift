# Artlift

Artlift is a platform that supports and cheers up artists by helping them build strong connections with fans and industry. Artists can upload artworks, showcase their creative journey, and interact with their audience. Fans can follow artists, leave comments, and support them through purchases or subscription plans.

---

## Features

- **Artist Profiles**
  - Logo, biography, passions, featured artworks, etc.
- **Artwork Display**
  - Image slider, titles, descriptions, comments, likes
- **Search**
  - Discover artists and artworks easily
- **Social Interactions**
  - Follow / Unfollow
  - Commenting on artworks
- **Admin Dashboard**
  - Manage profile
  - Upload artworks
  - Handle revenue & shipping data (future)
- **Purchasing**
  - Purchase artworks or subscriptions (future roadmap)

## Getting Started

### Frontend Setup

1. **Navigate to the `frontend/` Directory**:
   ```bash
   cd frontend/artlift-client
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run npm**:
   ```bash
   npm run dev
   ```
